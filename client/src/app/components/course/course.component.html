Data Structure Introduction to data structures and algorithms with C++

In this course, students learn structures and methods for efficiently storing/retrieving/processing data. Following topics will be covered in this course:

- Basic concepts of object-oriented programming using C++
- Stack, queue, linked list, tree, graph
- Sorting
- Hashing

Prerequisite: C programming experience

2020 Spring 2021 Spring Computer Vision Introduction to computer vision and machine learning with Python

This course aims to help students learn skills to process and analyze data using Python, which is widely used in various fields. In this course, students learn how to use numpy and OpenCV library, and use them to implement a simple object recognition program.

Prerequisite: Python programming experience

2020 Fall Web Programming Introduction to basic web programming with Node.js and JavaScript

This course introduces Node.js platform, which is widely used for client and server development. It starts from learning basic JavaScript programming skills and students will use it to implement simple clients. A number of popular packages for web framework (e.g., express) and database management (e.g., mongoose) will be covered and will be used to implement a backend server.

Prerequisite: HTML/CSS
